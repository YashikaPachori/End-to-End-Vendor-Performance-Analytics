# Logs

This folder contains log files generated during the execution of the ETL pipeline and data processing scripts.

## Included Log Files

* **ingestion.log** – Records the data ingestion process, including dataset loading, chunk-wise insertion into the SQLite database, execution status, processing time, and any errors encountered.
* **vendor_summary.log** – Captures the execution of the vendor summary generation process, including data aggregation, feature engineering, and summary creation.

These log files help monitor the ETL workflow, simplify debugging, and provide a record of the successful execution of different stages of the data pipeline.
